# Agricultural Monitoring System

## Overview
Autonomous agricultural monitoring rover designed to navigate through crops without causing damage while collecting environmental data for harvest optimization.

## Project Details
- **Duration**: December 2021 - March 2022
- **Team Size**: 4 members
- **Achievement**: Top 10 selection among 500 ideas at national-level ideathon competition

## Key Features
- **Arch-Shaped Chassis**: Non-disruptive movement through crop rows
- **Autonomous Navigation**: Raspberry Pi-based control with L298N motor driver
- **Sensor Fusion**: Integrates soil moisture and humidity data for harvest timing optimization
- **IoT Connectivity**: Remote monitoring and control capabilities

## Technical Stack
**Hardware**
- Raspberry Pi
- L298N Motor Driver
- Soil Moisture Sensors
- Humidity Sensors

**Software**
- Python (control & sensor integration)
- IoT protocols

**Design Tools**
- Fusion 360 (3D modeling)
- 3D printing
